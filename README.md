# AI-Powered Workforce Analytics & Talent Intelligence Dashboard

An intelligent workforce analytics solution designed to analyze employee data, identify attrition trends, evaluate workforce patterns, and support data-driven talent decisions.

## Project Tasks

- Model Research
- Python Workforce Analytics
- SQL Workforce Analytics

## Technologies Used

- Python
- SQL
- Machine Learning
- Pandas
- Matplotlib
- Scikit-learn
- SQLite
- Google Colab

## Key Features

- Employee attrition analysis
- Department-wise workforce insights
- Job satisfaction analysis
- Attrition risk prediction using Random Forest
- Feature importance analysis
- SQL-based workforce analytics
- High attrition-risk indicator analysis
